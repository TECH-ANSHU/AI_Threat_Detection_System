# Screenshots Directory (`screenshots/`)

## Purpose
This directory is reserved for hosting high-resolution screenshots and video demonstrations of the Security Operations Center (SOC) dashboard. These assets are referenced directly in the main repository `README.md` to showcase the application interface on GitHub and portfolio websites.

## Recommended Screenshots to Add
To present a professional showcase for job applications (e.g. Deloitte, cyber agencies), capture and place the following images in this directory:
1. **`soc_dashboard_live.png`**: A screenshot of the **Live SOC Monitor** tab showing the live network traffic feed with blinking status indicators, alert rates, and SIEM logs updating.
2. **`threat_analytics.png`**: A screenshot of the **Threat Analytics** page displaying the donut charts, horizontal bar graphs of attack categories, and the line graph showing threat rates.
3. **`ml_diagnostics.png`**: A screenshot of the **ML Model Center** showcasing the Confusion Matrix heatmaps, feature importance charts, and classifier parameters.
4. **`analyst_action_desk.png`**: A screenshot of the **Alert Ticket System** demonstrating how analysts search, filter, update ticket statuses, insert notes, and download CSV reports.

## System Interaction
- The main `README.md` links directly to files in this folder to display them in the markdown preview.
- These files are static assets and are not loaded by the Python runtime.
